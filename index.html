Производимое оборудование

Линии волочения

Предназначена для волочения катанки в проволоку.

Линии бронирования

Линия бронирования кабеля — современное высокотехнологичное оборудование, применяемое для обмотки кабельной заготовки стальными лентами

Лентообмоточное оборудование

Обмоточные машины предназначены для изолирования жилы круглого или прямоугольного сечения путем обмотки ее изоляционными материалами.

Линии бухтования

Линия бухтования кабеля предназначена для намотки поступающего кабеля в бухты определенного метража. Снятие бухты с бухтодержателя происходит вручную.

Крутильные машины

Данное оборудования предназначено для скрутки как изолированной, так не изолированной жилы различных сечений и формы.

Плетельное оборудование (Оплёточное оборудование)

Оборудование предназначено для нанесения оплётки на кабель.

Оборудования для разделки и переработки кабельно-проводниковой продукции

Оборудование для переработки и разделки кабеля предназначено для извлечения алюминия, а также меди из кабельно-проводниковой продукции путем измельчения исходного сырья на гранулы.

Линия выплавки катанки

Линия предназначена для непрерывного производства бескислородной меди Ф17/14 медной катанки

Дополнительное оборудование

ЗАСИ, отдающие устройства, приёмные устройства.
